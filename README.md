# VideoWallpaper  
A demo of video wallpaper for python3  

## 使用  
1. 执行`pip install pypiwin32`以安装所需的库  
2. 使用前先将`ffplay.exe`放入ffmpeg-win64文件夹内（也可在VideoWallpaper.py中修改）
